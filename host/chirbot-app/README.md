# chirbot-app

> **Status:** planned — no code yet; stack not selected (cross-platform assumed).

PC application for CHIRBot. Planned capabilities:

- Matrix and module configuration over USB
- Virtual input streaming to/from the core (e.g., remote input, online input
  visualization for streaming)
- Realtime input visualization
- Replay/macro management for recordings on the device's microSD (TASD files)
